# User Registration System

## 📌 Project Overview
A User Registration System built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

## 🚀 Features
- User Registration
- GET & POST REST APIs
- MongoDB Integration
- Duplicate Email Validation
- Registered Users List
- Total Users Counter
- Responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## 📂 Project Structure
- server.js
- controllers/
- routes/
- models/
- config/
- public/

## ▶️ Run Locally

```bash
npm install
npm start
```

Server runs on:

```
http://localhost:3000
```
